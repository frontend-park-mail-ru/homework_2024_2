Methods

set(obj, path, value) → {Object}

Устанавливает значение в указанное свойство объекта, используя путь.
Parameters:
Name Type Description
obj Object Объект, в который нужно установить значение.
path string Путь к свойству объекта в виде строки. Путь может быть задан в виде 'a.b.c' или 'a[0].b["key"]'.
value * Значение, которое необходимо установить.
Source:
  • set.js, line 36
Throws:
Если пытаются обратиться к элементу массива с отрицательным индексом.
Type
Error
Returns:
Объект с установленным значением.
Type
Object
Examples
const obj = {};
set(obj, 'a.b.c', 42);
// obj теперь равно { a: { b: { c: 42 } } }
const obj = {};
set(obj, 'a[0].b["key"]', 'value');
// obj теперь равно { a: [ { b: { key: 'value' } } ] }
